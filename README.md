## Structure

Each session folder contains:
- `tex/` — LaTeX technical report
- `slides/` — presentation deck outline
- `notes.md` — session notes
- `resources.md` — curated tools, repos, datasets
- `reading/` — annotated reading list
- `experiments/` — demo notebooks & Python scripts

## Website

The [`docs/`](docs/) folder powers a [GitHub Pages site](https://prabakaranc98.github.io/Thursday-Learning-Hours) that lists all sessions with links to reading lists, slides, notes, experiments, and YouTube recordings.
