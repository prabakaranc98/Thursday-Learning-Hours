# Thursday Learning Hours 🧠📅

> **52 Thursdays · 52 Topics · 1 Year of AI/ML/DS**
>
> 🌐 **Site:** [prabakaranc98.github.io/Thursday-Learning-Hours](https://prabakaranc98.github.io/Thursday-Learning-Hours)

A personal initiative to run a weekly seminar, tutorial, or hands-on demo every Thursday for one year — covering **Foundations**, **Frontiers**, and **Frameworks** across Artificial Intelligence, Machine Learning, and Data Science Research & Engineering.

Each session contains:
- 📝 A topic overview with learning objectives
- 💻 A working Jupyter notebook demo / experiment
- 📚 Curated references and further reading

## 🚀 Getting Started

```bash
# Clone the repository
git clone https://github.com/prabakaranc98/Thursday-Learning-Hours.git
cd Thursday-Learning-Hours

# Create a virtual environment (Python 3.10+)
python -m venv .venv
source .venv/bin/activate   # Windows: .venv\Scripts\activate

# Install common dependencies
pip install numpy pandas matplotlib seaborn scikit-learn jupyter

# Launch Jupyter
jupyter notebook
```
