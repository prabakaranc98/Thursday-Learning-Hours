# Thursday Learning Hours

Thursday Learning Hours is now a filesystem-first working archive for small,
disciplined technical sessions. This repo is not organized as a hosted website.
Each TLH has its own folder with notes, links, readings, slides, and experiment
work.

## Repository Discipline

- Keep one folder per session: `TLH-N-Short-Topic/`.
- Keep the root [`sessions.json`](sessions.json) as the machine-readable index.
- Put runnable work under `experiments/`.
- Put discussion notes in `notes.md`.
- Put papers in `reading/README.md`.
- Put links and tools in `links.md` and `resources.md`.
- Do not add static-site scaffolding, generated web assets, or GitHub Pages
  hosting files.
